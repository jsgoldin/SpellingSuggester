A program that checks the spelling of words and generates suggestions for mispellings


Contents:
    src/
        Dictionary.java - trie data structure for dictionary an methods for performing checks and suggestions
        Node.java - A component of the trie dictionary
        Driver.java - A simple program to test out the dictionary

    words_ospd.txt - contains all valid words in the dictionary
    LanguageModel.txt - maps dictionary words to there corresponding estimated frequency in the English language
